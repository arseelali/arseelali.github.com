Instructions

  1. Enter a YouTube link in the following format:

    https://www.youtube.com/watch?v=XXXXXXXXXXX

    If your link does not have https://www. at the beginning, please add it.

  2. Paste the link and click 'Generate.' To use this iFrame, go to Schoology, right click and click "Inspect" on any blank space on the website.
  3. In the opened panel, look for an element. When you hover on it, that element on the page should turn blue. Now right click on that element.
  4. Then click Edit as HTML. Select everything in the box and delete it.
  5. Paste the code provided and click the X icon at the top right of the panel.
